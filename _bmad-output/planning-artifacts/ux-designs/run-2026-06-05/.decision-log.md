## Decisions
1. **Surface Selection:** The Web/App interface maps to the **Admin Cockpit**, which necessitates the `data-surface="app"` setting (high density, 13.5px body, minimal motion).
2. **User Journeys:** Extracted two primary journeys mapping to the architectural components:
   - System Admin using the React Dashboard (Command Center, Crawler Terminal, Node Explorer).
   - Investor using the Telegram Bot for risk alerts and Q&A.
3. **Typographic Adherence:** We are strictly utilizing JetBrains Mono for the Crawler Terminal logs and entity IDs, and Be Vietnam Pro for standard UI labels, ensuring the VN-first technical aesthetic is met.
4. **Color Budget:** Restricted to signal accents (`--accent-freq: signal`). Colors are used almost exclusively for status (Mint = Live, Gap = Error) rather than decoration.
