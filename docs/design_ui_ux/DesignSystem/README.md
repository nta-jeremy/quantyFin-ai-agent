# QuantyFin Design System

Brand identity + production-ready, Tailwind v4 (shadcn-standard) component library for **QuantyFin** — one token foundation across four surfaces (marketing · application · portal · store).

**This repo keeps three docs:**
- **[`DESIGN.md`](./DESIGN.md)** — the design system: brand, tokens, architecture, surfaces, principles, the Tailwind v4 layer, components, and **how to consume it in a real repo**.
- **[`SKILL.md`](./SKILL.md)** — the invocable agent skill (how to generate interfaces with the DS correctly).
- **[`CLAUDE.md`](./CLAUDE.md)** — how to work in this repo + project state/history.
